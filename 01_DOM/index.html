Getting HTML Elements with DOM Methods

Hello World!

Get element by id

var element = document.getElementById("demo"); console.log(element);

Get element by tag name

var elements = document.getElementsByTagName("p"); console.log(elements);

Get element by class name

var elements = document.getElementsByClassName("demo-class"); console.log(elements);

Get element by query selector

var element = document.querySelector("#demo"); console.log(element);

Get element by query selector all

var elements = document.querySelectorAll(".demo"); console.log(elements);

Getting HTML Elements with DOM Attributes

DOM Attributes

element.getAttributes("name") element.setAttributes("name", "value")

Attribute Presence vs value

bin.getAttribute("disabled") // true
bin.setAttribute("disabled") // "disabled"
bin.getAttribute("class") // "bin"
bin.setAttribute("class", "bin") // "bin"
bin.removeAttribute("class") // null
bin.hasAttribute("class") // false
bin.hasAttribute("disabled") // true
bin.deleteAttribute("disabled") // null
